Instrukcja uruchomienia aplikacji:
1. Zainstalować XAMPP
2. Wypakować zawartość do XAMPP/htdocs (Przykładowa ścieżka index.php: ../XAMPP/htdocs/lic/index.php)
3. Uruchomić XAMPP Control Panel a w nim Apache oraz MySQL
4. W adresie przeglądarki wpisać: localhost/phpmyadmin/
5. Utworzyć bazę o nazwie 'uzytkownicy'
6. Importować dane do bazy z pliku lic/baza/uzytkownicy.sql
7. W adresie przeglądarki wpisać localhost/lic
8. Aplikacja działa!
